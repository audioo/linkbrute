## Sites
- Ghostbin.co
- Rentry.co
- Pasteall.org
- Paste.in
- Paste.ee

## Usage
In the netflip application, click on any site in order to launch. You can run multiple sites at a time. Click the *Stop All* button to terminate all threads.
